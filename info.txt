# 3️⃣ Database Design (MySQL)

Create a database first.

```sql
CREATE DATABASE hotel_management;
USE hotel_management;
```

---

## Table 1: Customers

```sql
CREATE TABLE customers (
    customer_id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    phone VARCHAR(15),
    email VARCHAR(100),
    address TEXT
);
```

---

## Table 2: Rooms

```sql
CREATE TABLE rooms (
    room_id INT AUTO_INCREMENT PRIMARY KEY,
    room_type VARCHAR(50),
    price DECIMAL(10,2),
    status VARCHAR(20)
);
```

Example values for status:

* Available
* Booked
* Maintenance

---

## Table 3: Reservations

```sql
CREATE TABLE reservations (
    reservation_id INT AUTO_INCREMENT PRIMARY KEY,
    customer_id INT,
    room_id INT,
    check_in DATE,
    check_out DATE,
    total_price DECIMAL(10,2),
    status VARCHAR(20),
    FOREIGN KEY (customer_id) REFERENCES customers(customer_id),
    FOREIGN KEY (room_id) REFERENCES rooms(room_id)
);
```

---

## Table 4: Payments

```sql
CREATE TABLE payments (
    payment_id INT AUTO_INCREMENT PRIMARY KEY,
    reservation_id INT,
    amount DECIMAL(10,2),
    payment_date DATE,
    method VARCHAR(50),
    FOREIGN KEY (reservation_id) REFERENCES reservations(reservation_id)
);
```

---

# 4️⃣ Connect Python to MySQL

Example connection code:

```python
import mysql.connector

conn = mysql.connector.connect(
    host="localhost",
    user="root",
    password="yourpassword",
    database="hotel_management"
)

cursor = conn.cursor()
```

Test query:

```python
cursor.execute("SELECT * FROM rooms")
for row in cursor:
    print(row)
```

# 7️⃣ Reservation Logic

### Room availability check

```sql
SELECT * FROM rooms WHERE status='Available';
```

---

### Price Calculation

Example:

```python
days = (checkout - checkin).days
total_price = room_price * days
```

---

### After Booking

Update room status:

```sql
UPDATE rooms
SET status='Booked'
WHERE room_id=?
```

---

# 8️⃣ Check-Out System

When customer checks out:

1. Calculate final bill
2. Insert payment record
3. Update reservation status
4. Set room to **Available**

Example:

```sql
UPDATE rooms
SET status='Available'
WHERE room_id=?
```
